Web and Mobile Developer

Hi! My name is Romeo Selwyn Villar. I associate myself with the name Panzerweb. I am a developer that really loves to create simple to complex projects.

Right now? Hmmm... I am working on expanding my skills in mobile development using Flutter. VueJS on frontend web development, as well as Django for backend development for both web and mobile platforms.

Fun Fact about me is that, I still struggle to read documentations, and other technologies which is frustrating knowing how essential it is. Moreover, I also play guitar, write stories, do UI/UX Designing, and graphic designs.
